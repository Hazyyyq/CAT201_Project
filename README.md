# CAT201_Project
A repository for CAT201 Project. This project features a dynamic webpage with a Java Tomcat backend and a React frontend.
## Kaki Gamerz Website Link
https://kakigamerz.vercel.app

## [STEP 1] Run Locally (backend)
To run the backend on your localhost:

1. Go to the `java-backend [backend]` directory
2. Go to Main.java which is inside `src/main/java/com.kakigamerz/Main`
3. Run the file to start the Tomcat server on `http://localhost:808`

## [STEP 2] Run Locally (frontend)
To run the frontend on your localhost:

1. Clone the repository or Download as ZIP files:
2. Open the terminal in the react-frontend directory
3. Install dependencies via terminal:
``` bash
npm install
```
4. Change the directory via terminal:
``` bash
cd react-frontend 
```
5. Run the dev via terminal:
``` bash
npm run dev 
```
6. Click the link of the local host
```bash
  VITE v7.3.0  ready in 3328 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```
7. Enjoy the Kaki Gamerz website
